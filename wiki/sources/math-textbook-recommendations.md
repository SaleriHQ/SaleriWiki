# Mathematics Textbook Recommendations

## 概述

来自 /sci/ 的数学教科书推荐清单，从高中预科数学到研究生级别，涵盖代数、微积分、线性代数、分析、拓扑等核心领域。

## 学习路径概览

```
预科数学 → 微积分 → 线性代数 → 抽象代数/分析
   ↓          ↓            ↓            ↓
Khan Academy  Apostol     Axler       Rudin/Lang
  Stewart    Spivak      Shilov      Hartshorne
```

## 推荐学习路径

### 数学爱好者路径

```
1. Khan Academy 预科 (中学数学)
2. Precalculus (Stewart/Axler)
3. Calculus (Apostol/Spivak)
4. Linear Algebra (Shilov + Axler)
5. ODE (Tenenbaum & Pollard)
6. Real Analysis (Rudin)
7. Abstract Algebra (Herstein/Artin)
```

### 数学专业学生路径

```
1. Proofs (Velleman/Smith)
2. Calculus + Multivariable (Apostol)
3. Linear Algebra (Hoffman & Kunze)
4. ODE/PDE (Haberman)
5. Complex Analysis (Stein & Shakarchi)
6. Real Analysis (Rudin)
7. Abstract Algebra (Artin + Dummit & Foote)
8. Topology (Munkres)
9. Algebraic Geometry (Hartshorne)
```

## 核心观点

> "True mathematics involves proofs, lots and lots of proofs."

**关键数学里程碑**：
1. 掌握证明技巧（证明书）
2. 学好单变量微积分
3. 理解线性代数（不止矩阵运算）
4. 学习抽象代数思维方式
5. 深入实分析
