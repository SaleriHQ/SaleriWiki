# 第4章 多项式

本章包含我们将要用来研究从向量空间到自身的线性映射的多项式材料。本章中的许多结果对你来说可能已经从其他课程中熟悉了；这里包含它们是为了完整性。因为本章不是关于线性代数的，你的老师可能快速讲解。你可能不需要仔细审查所有证明。但是，请确保你至少阅读并理解本章中所有结果的陈述——它们将在后面的章节中使用。

本章首先简要讨论复数的代数性质。然后我们证明非常数多项式不可能有超过其次数的零点。我们还给出了多项式除法算法的一个基于线性代数的证明，即使你已经熟悉不使用线性代数的证明，这也值得一读。

正如我们将看到的，代数学基本定理导致每个多项式在标量域为 $\mathbf{C}$ 时分解为一次因式，或在标量域为 $\mathbf{R}$ 时分解为至多二次的因式。

**本章的持续假设：**
- $\mathbf{F}$ 表示 $\mathbf{R}$ 或 $\mathbf{C}$。

---

## 4A 复数

在讨论具有复系数或实系数的多项式之前，我们需要更多地了解复数。

**4.1 定义：实部、虚部**

设 $z = a + bi$，其中 $a$ 和 $b$ 是实数。

- $z$ 的**实部**，记为 $\operatorname{Re} z$，定义为 $\operatorname{Re} z = a$。
- $z$ 的**虚部**，记为 $\operatorname{Im} z$，定义为 $\operatorname{Im} z = b$。

因此对于每个复数 $z$，我们有

$$z = \operatorname{Re} z + (\operatorname{Im} z)i.$$

**4.2 定义：复共轭、模长**

设 $z \in \mathbf{C}$。

- $z \in \mathbf{C}$ 的**复共轭**，记为 $\bar{z}$，定义为

$$\bar{z} = \operatorname{Re} z - (\operatorname{Im} z)i.$$

- 复数 $z$ 的**模长**，记为 $|z|$，定义为

$$|z| = \sqrt{(\operatorname{Re} z)^2 + (\operatorname{Im} z)^2}.$$

**4.3 例：** 实部与虚部、复共轭、模长

设 $z = 3 + 2i$。则

- $\operatorname{Re} z = 3$ 且 $\operatorname{Im} z = 2$；
- $\bar{z} = 3 - 2i$；
- $|z| = \sqrt{3^2 + 2^2} = \sqrt{13}$。

通过识别复数 $z \in \mathbf{C}$ 与有序对 $(\operatorname{Re} z, \operatorname{Im} z) \in \mathbf{R}^2$，我们可以将 $\mathbf{C}$ 与 $\mathbf{R}^2$ 等同。注意 $\mathbf{C}$ 是一维复向量空间（与 $\mathbf{R}^2$ 等同时是二维实向量空间），但我们也可以将 $\mathbf{C}$ 看作二维实向量空间。

每个复数的模长是一个非负数。具体地，如果 $z \in \mathbf{C}$，则 $|z|$ 等于 $\mathbf{R}^2$ 中原点到点 $(\operatorname{Re} z, \operatorname{Im} z)$ 的距离。

你应该验证当且仅当 $z$ 是实数时 $z = \bar{z}$，以及复共轭和模长具有下述多部分结果中列出的性质。

**4.4 复数的性质**

设 $w, z \in \mathbf{C}$。则下列等式和不等式成立。

| 性质                 | 表达式                                                                       |                     |      |     |       |                     |      |     |     |
| ------------------ | ------------------------------------------------------------------------- | ------------------- | ---- | --- | ----- | ------------------- | ---- | --- | --- |
| $z$ 与 $\bar{z}$ 的和 | $z + \bar{z} = 2\operatorname{Re} z$                                      |                     |      |     |       |                     |      |     |     |
| $z$ 与 $\bar{z}$ 的差 | $z - \bar{z} = 2(\operatorname{Im} z)i$                                   |                     |      |     |       |                     |      |     |     |
| $z$ 与 $\bar{z}$ 的积 | $z\bar{z} =                                                               | z                   | ^2$  |     |       |                     |      |     |     |
| 复共轭的可加性和乘积性        | $\overline{w + z} = \bar{w} + \bar{z}$ 且 $\overline{wz} = \bar{w}\bar{z}$ |                     |      |     |       |                     |      |     |     |
| 双重复共轭              | $\bar{\bar{z}} = z$                                                       |                     |      |     |       |                     |      |     |     |
| 实部和虚部被模长界定         | $                                                                         | \operatorname{Re} z | \leq | z   | $ 且 $ | \operatorname{Im} z | \leq | z   | $   |
| 复共轭的模长             | $                                                                         | \bar{z}             | =    | z   | $     |                     |      |     |     |
| 模长的乘积性             | $                                                                         | wz                  | =    | w   |       | z                   | $    |     |     |
| 三角不等式              | $                                                                         | w + z               | \leq | w   | +     | z                   | $    |     |     |

**证明** 除了最后一项，上述断言的常规验证留给读者。为了验证三角不等式，我们有

$$
\begin{aligned}
|w + z|^2 &= (w + z)(\bar{w} + \bar{z}) \\
&= w\bar{w} + z\bar{z} + w\bar{z} + z\bar{w} \\
&= |w|^2 + |z|^2 + w\bar{z} + \overline{w\bar{z}} \\
&= |w|^2 + |z|^2 + 2\operatorname{Re}(w\bar{z}) \\
&\leq |w|^2 + |z|^2 + 2|w\bar{z}| \\
&= |w|^2 + |z|^2 + 2|w||z| \\
&= (|w| + |z|)^2.
\end{aligned}
$$

两边取平方根即得所需不等式。$\blacksquare$

参见习题2关于逆三角不等式 $|w| - |z| \leq |w - z|$。

## 4B 多项式的零点

回顾：一个函数 $p : \mathbf{F} \rightarrow \mathbf{F}$ 称为**次数为 $m$ 的多项式**，如果存在 $a_0, a_1, \ldots, a_m \in \mathbf{F}$ 且 $a_m \neq 0$，使得

$$p(z) = a_0 + a_1 z + \cdots + a_m z^m$$

对所有 $z \in \mathbf{F}$ 成立。如果上述形式的多项式表示不唯一，则一个多项式可能有多于一个次数。我们的首要任务是证明这不可能发生。

方程 $p(z) = 0$ 的解在研究 $p \in \mathcal{P}(\mathbf{F})$ 中起着关键作用。因此这些解有一个特殊名称。

**4.5 定义：多项式的零点**

设 $\lambda \in \mathbf{F}$。如果 $p(\lambda) = 0$，则称 $\lambda$ 为多项式 $p \in \mathcal{P}(\mathbf{F})$ 的**零点**（或根）。

下一个结果是证明多项式次数唯一性的关键工具。

**4.6 每个多项式的零点对应一个一次因式**

设 $m$ 是正整数且 $p \in \mathcal{P}(\mathbf{F})$ 是次数为 $m$ 的多项式。

设 $\lambda \in \mathbf{F}$。则 $p(\lambda) = 0$ 当且仅当存在一个次数为 $m - 1$ 的多项式 $q \in \mathcal{P}(\mathbf{F})$，使得

$$p(z) = (z - \lambda)q(z)$$

对所有 $z \in \mathbf{F}$ 成立。

**证明** 首先假设 $p(\lambda) = 0$。设 $a_0, a_1, \ldots, a_m \in \mathbf{F}$ 使得

$$p(z) = a_0 + a_1 z + \cdots + a_m z^m$$

对所有 $z \in \mathbf{F}$ 成立。则

$$p(z) - p(\lambda) = a_1(z - \lambda) + \cdots + a_m(z^m - \lambda^m).$$

对于每个 $k \in \{1, \ldots, m\}$，方程

$$z^k - \lambda^k = (z - \lambda)\sum_{j=1}^{k} \lambda^{j-1}z^{k-j}$$

表明 $z^k - \lambda^k$ 等于 $z - \lambda$ 乘以某个次数为 $k - 1$ 的多项式。因此上述等式表明 $p$ 等于 $z - \lambda$ 乘以某个次数为 $m - 1$ 的多项式，正如所需。

为了证明另一个方向的蕴含关系，现在假设存在一个多项式 $q \in \mathcal{P}(\mathbf{F})$ 使得 $p(z) = (z - \lambda)q(z)$ 对所有 $z \in \mathbf{F}$ 成立。则

$$p(\lambda) = (\lambda - \lambda)q(\lambda) = 0,$$

正如所需。$\blacksquare$

现在我们可以证明多项式不会有太多零点。

**4.8 次数为 $m$ 意味着最多 $m$ 个零点**

设 $m$ 是正整数且 $p \in \mathcal{P}(\mathbf{F})$ 是次数为 $m$ 的多项式。则 $p$ 在 $\mathbf{F}$ 中最多有 $m$ 个零点。

**证明** 我们对 $m$ 使用数学归纳法。如果 $m = 1$，则所需结果成立，因为如果 $a_1 \neq 0$，则多项式 $a_0 + a_1 z$ 只有一个零点（等于 $-a_0/a_1$）。

因此假设 $m > 1$ 且对 $m - 1$ 的情况结论成立。

如果 $p$ 在 $\mathbf{F}$ 中没有零点，则所需结果成立，我们完成证明。因此假设 $p$ 有一个零点 $\lambda \in \mathbf{F}$。由4.6，存在一个次数为 $m - 1$ 的多项式 $q \in \mathcal{P}(\mathbf{F})$，使得

$$p(z) = (z - \lambda)q(z)$$

对所有 $z \in \mathbf{F}$ 成立。我们的归纳假设表明 $q$ 在 $\mathbf{F}$ 中最多有 $m - 1$ 个零点。上式表明 $p$ 在 $\mathbf{F}$ 中的零点正好是 $q$ 在 $\mathbf{F}$ 中的零点加上 $\lambda$。因此 $p$ 在 $\mathbf{F}$ 中最多有 $m$ 个零点。$\blacksquare$

上述结果意味着多项式的系数是唯一确定的（因为如果一个多项式有两组不同的系数，则将多项式的两个表示相减会得到一个具有某些非零系数但有无穷多零点多项式）。特别是，多项式的次数是唯一定义的。

我们规定零多项式的次数为 $-\infty$。当各种合理结果需要使用 $-\infty$ 时，例如 $-\infty < m$ 和 $\deg(pq) = \deg p + \deg q$，将次数定义为 $-\infty$ 可以避免不必要的特殊情况。

## 4C 多项式的除法算法

如果 $p$ 和 $s$ 是非负整数，且 $s \neq 0$，则存在唯一的非负整数 $q$ 和 $r$，使得

$$p = sq + r$$

且 $0 \leq r < s$。考虑用 $s$ 去除 $p$，得到商 $q$ 和余数 $r$。我们的下一个结果给出了多项式的类似结果。因此，这个结果通常被称为多项式除法算法，尽管如本文所述，它实际上不是一个算法，只是一个有用的结果。

多项式除法算法可以用线性代数来证明。适当于线性代数教科书，这里给出的证明使用了线性代数技术，并很好地利用了 $\mathcal{P}_n(\mathbf{F})$（系数在 $\mathbf{F}$ 中、次数至多 $n$ 的多项式构成的 $n+1$ 维向量空间）的一个基。

**4.9 多项式的除法算法**

假设 $p, s \in \mathcal{P}(\mathbf{F})$，且 $s \neq 0$。则存在唯一的多项式 $q, r \in \mathcal{P}(\mathbf{F})$，使得

$$p = sq + r$$

且 $\deg r < \deg s$。

**证明** 设 $n = \deg p$ 且 $m = \deg s$。如果 $n < m$，则取 $q = 0$ 且 $r = p$，得到所需方程 $p = sq + r$ 且 $\deg r < \deg s$。因此我们现在假设 $n \geq m$。

列表

$$1, z, \ldots, z^{m-1}, s, zs, \ldots, z^{n-m}s$$

在 $\mathcal{P}_n(\mathbf{F})$ 中是线性无关的，因为这个列表中的每个多项式次数不同。此外，列表4.10的长度为 $n + 1$，等于 $\dim \mathcal{P}_n(\mathbf{F})$。因此由2.38，列表4.10是 $\mathcal{P}_n(\mathbf{F})$ 的一个基。

因为 $p \in \mathcal{P}_n(\mathbf{F})$ 且4.10是 $\mathcal{P}_n(\mathbf{F})$ 的一个基，存在唯一的常数 $a_0, a_1, \ldots, a_{m-1} \in \mathbf{F}$ 和 $b_0, b_1, \ldots, b_{n-m} \in \mathbf{F}$，使得

$$
\begin{aligned}
p &= a_0 + a_1 z + \cdots + a_{m-1}z^{m-1} + s(b_0 + b_1 z + \cdots + b_{n-m}z^{n-m}) \\
&= \underbrace{(a_0 + a_1 z + \cdots + a_{m-1}z^{m-1})}_{r} + \underbrace{s(b_0 + b_1 z + \cdots + b_{n-m}z^{n-m})}_{sq}.
\end{aligned}
$$

如上定义 $r$ 和 $q$，我们看到 $p$ 可以写成 $p = sq + r$，且 $\deg r < \deg s$，正如所需。

满足这些条件的 $q, r \in \mathcal{P}(\mathbf{F})$ 的唯一性来自于满足4.11的常数 $a_0, a_1, \ldots, a_{m-1} \in \mathbf{F}$ 和 $b_0, b_1, \ldots, b_{n-m} \in \mathbf{F}$ 的唯一性。$\blacksquare$

## 4D 复数域上的多项式因式分解

我们一直在同时处理具有复系数和实系数的多项式，用 $\mathbf{F}$ 表示 $\mathbf{R}$ 或 $\mathbf{C}$。现在我们将看到这两种情况之间的差异。

代数学基本定理是一个存在性定理。它的证明不会导出寻找零点的算法。二次公式明确给出了二次多项式的零点。对于三次和四次多项式，也存在类似的公式。对于五次及更高次的多项式，不存在这样的公式。

我们的代数学基本定理证明隐含地使用了连续实值函数在 $\mathbf{R}^2$ 中闭盘上达到最小值的结果。你可以搜索到代数学基本定理的许多其他证明。如果你对解析函数比较熟悉，使用刘维尔定理的证明特别简洁。代数学基本定理的所有证明都需要一些分析，因为如果用一组例如形如 $c + di$（其中 $c, d$ 是有理数）的数来替代 $\mathbf{C}$，结果就不成立了。

**4.12 代数学基本定理，第一版本**

每个具有复系数的非常数多项式在 $\mathbf{C}$ 中都有一个零点。

**证明** 德·莫弗尔定理，你可以通过对 $k$ 使用归纳法并利用余弦和正弦的加法公式来证明，它表明如果 $k$ 是正整数且 $\theta \in \mathbf{R}$，则

$$(\cos\theta + i\sin\theta)^k = \cos k\theta + i\sin k\theta.$$

假设 $w \in \mathbf{C}$ 且 $k$ 是正整数。利用极坐标，我们知道存在 $r \geq 0$ 和 $\theta \in \mathbf{R}$，使得

$$w = r(\cos\theta + i\sin\theta).$$

德·莫弗尔定理意味着

$$\left(r^{1/k}(\cos\theta + i\sin\theta)\right)^k = w.$$

因此每个复数都有 $k$ 次根，这个事实我们将很快用到。

设 $p$ 是一个具有复系数的非常数多项式，其最高次非零项为 $c_m z^m$。则当 $|z| \rightarrow \infty$ 时 $|p(z)| \rightarrow \infty$（因为 $|p(z)|/|z|^m \rightarrow |c_m|$ 当 $|z| \rightarrow \infty$）。因此连续函数 $z \mapsto |p(z)|$ 在 $\mathbf{C}$ 中的某个点 $\zeta$ 处有全局最小值。为了证明 $p(\zeta) = 0$，假设 $p(\zeta) \neq 0$。

通过

$$q(z) = \frac{p(z + \zeta)}{p(\zeta)}$$

定义一个新多项式 $q$。函数 $z \mapsto |q(z)|$ 在 $z = 0$ 处有全局最小值 $1$。写成

$$q(z) = 1 + a_k z^k + \cdots + a_m z^m,$$

其中 $k$ 是使得 $z^k$ 系数非零的最小正整数；换句话说，$a_k \neq 0$。

设 $\beta \in \mathbf{C}$ 使得 $\beta^k = -1/a_k$。存在常数 $c > 1$，使得如果 $t \in (0, 1)$，则

$$|q(t\beta)| \leq |1 + a_k t^k \beta^k| + t^{k+1}c = 1 - t^k(1 - tc).$$

因此在上式中取 $t = 1/(2c)$，我们有 $|q(t\beta)| < 1$，这与 $z \mapsto |q(z)|$ 的全局最小值为 $1$ 的假设矛盾。这个矛盾意味着 $p(\zeta) = 0$，表明 $p$ 有一个零点，正如所需。$\blacksquare$

计算机可以使用巧妙的数值方法找到任何多项式零点的良好近似，即使无法找到精确零点。例如，没有人会给出一个定义为

$$p(x) = x^5 - 5x^4 - 6x^3 + 17x^2 + 4x - 7$$

的多项式零点的精确公式。然而，计算机可以发现 $p$ 的零点大约是 $-1.87, -0.74, 0.62, 1.47, 5.51$ 这五个数。

代数学基本定理的第一版本导致以下关于复系数多项式的因式分解结果。注意在这个因式分解中，$p$ 的零点是 $\lambda_1, \ldots, \lambda_m$，这些是使下述结果中等式右边等于 $0$ 的 $z$ 的唯一值。

**4.13 代数学基本定理，第二版本**

如果 $p \in \mathcal{P}(\mathbf{C})$ 是一个非常数多项式，则 $p$ 有唯一的（因式顺序除外）如下形式的因式分解

$$p(z) = c(z - \lambda_1) \cdots (z - \lambda_m),$$

其中 $c, \lambda_1, \ldots, \lambda_m \in \mathbf{C}$。

**证明** 设 $p \in \mathcal{P}(\mathbf{C})$ 且 $m = \deg p$。我们对 $m$ 使用归纳法。如果 $m = 1$，则所需的因式分解存在且唯一。因此假设 $m > 1$ 且对所有次数为 $m - 1$ 的多项式所需的因式分解存在且唯一。

首先我们将证明 $p$ 的所需因式分解存在。由代数学基本定理第一版本（4.12），$p$ 有一个零点 $\lambda \in \mathbf{C}$。由4.6，存在一个次数为 $m - 1$ 的多项式 $q$，使得

$$p(z) = (z - \lambda)q(z)$$

对所有 $z \in \mathbf{C}$ 成立。我们的归纳假设表明 $q$ 有所需的因式分解，代入上式即得 $p$ 的所需因式分解。

现在转向唯一性问题。常数 $c$ 被唯一地确定为 $p$ 中 $z^m$ 的系数。因此我们只需要证明除了顺序外，$\lambda_1, \ldots, \lambda_m$ 的选择是唯一的。如果

$$(z - \lambda_1) \cdots (z - \lambda_m) = (z - \tau_1) \cdots (z - \tau_m)$$

对所有 $z \in \mathbf{C}$ 成立，则因为当 $z = \lambda_1$ 时左边等于 $0$，所以右边某个 $\tau$ 等于 $\lambda_1$。重新标记，我们可以假设 $\tau_1 = \lambda_1$。现在如果 $z \neq \lambda_1$，我们可以将上式两边除以 $z - \lambda_1$，得到

$$(z - \lambda_2) \cdots (z - \lambda_m) = (z - \tau_2) \cdots (z - \tau_m)$$

对所有 $z \in \mathbf{C}$ 除了可能 $z = \lambda_1$ 都成立。实际上，上式对所有 $z \in \mathbf{C}$ 都成立，因为否则通过从左边减去右边我们会得到一个具有无穷多零点的非零多项式。上式和我们的归纳假设意味着除了顺序外，$\lambda$ 与 $\tau$ 相同，完成唯一性的证明。$\blacksquare$

## 4E 实数域上的多项式因式分解

具有实系数的多项式可能没有实零点。例如，多项式 $1 + x^2$ 没有实零点。

实数域上代数学基本定理的失败解释了实数向量空间和复数向量空间上线性代数之间的差异，我们将在后面的章节中看到。

为了在 $\mathbf{R}$ 上获得多项式因式分解定理，我们将使用我们在 $\mathbf{C}$ 上的因式分解定理。我们从以下结果开始。

**4.14 具有实系数的多项式有成对出现的非实零点**

设 $p \in \mathcal{P}(\mathbf{C})$ 是具有实系数的多项式。如果 $\lambda \in \mathbf{C}$ 是 $p$ 的零点，则 $\bar{\lambda}$ 也是 $p$ 的零点。

**证明** 因为 $p$ 有实系数，我们可以写成

$$p(z) = a_0 + a_1 z + \cdots + a_m z^m,$$

其中 $a_0, \ldots, a_m$ 是实数。假设 $\lambda \in \mathbf{C}$ 是 $p$ 的零点。则

$$a_0 + a_1 \lambda + \cdots + a_m \lambda^m = 0.$$

对这个等式两边取复共轭，利用复共轭的基本性质（见4.4），得到

$$a_0 + a_1 \bar{\lambda} + \cdots + a_m \bar{\lambda}^m = 0.$$

上式表明 $\bar{\lambda}$ 是 $p$ 的零点。$\blacksquare$

我们希望得到具有实系数的多项式的因式分解定理。我们从以下结果开始。

**4.15 二次多项式的因式分解**

设 $b, c \in \mathbf{R}$。则存在如下形式的多项式因式分解

$$x^2 + bx + c = (x - \lambda_1)(x - \lambda_2)$$

其中 $\lambda_1, \lambda_2 \in \mathbf{R}$，当且仅当 $b^2 \geq 4c$。

**证明** 首先假设 $b^2 < 4c$。则

$$x^2 + bx + c = \left(x + \frac{b}{2}\right)^2 + \left(c - \frac{b^2}{4}\right).$$

上式右端对每个 $x \in \mathbf{R}$ 都是正的。因此 $x^2 + bx + c$ 没有实零点，从而不能因式分解为 $(x - \lambda_1)(x - \lambda_2)$ 且 $\lambda_1, \lambda_2 \in \mathbf{R}$ 的形式。

反之，假设 $b^2 \geq 4c$。则存在实数 $d$ 使得 $d^2 = \frac{b^2}{4} - c$。由上式，

$$
\begin{aligned}
x^2 + bx + c &= \left(x + \frac{b}{2}\right)^2 - d^2 \\
&= \left(x + \frac{b}{2} + d\right)\left(x + \frac{b}{2} - d\right),
\end{aligned}
$$

这给出了所需的因式分解。$\blacksquare$

下一个结果给出了 $\mathbf{R}$ 上多项式的因式分解。其证明思路是使用代数学基本定理的第二版本（4.13），它给出了 $p$ 作为复系数多项式的因式分解。$p$ 的非实复零点成对出现；见4.14。因此，如果 $p$ 作为 $\mathcal{P}(\mathbf{C})$ 中元素的因式分解包含形如 $(x - \lambda)$ 的项，其中 $\lambda$ 是非实复数，则 $(x - \bar{\lambda})$ 也是因式分解中的一项。将这两项相乘，我们得到

$$x^2 - 2(\operatorname{Re} \lambda)x + |\lambda|^2,$$

这是一个所需形式的二次项。

上文概述的想法几乎提供了我们所需因式分解的存在性证明。然而，我们需要小心一点。假设 $\lambda$ 是一个非实复数且 $(x - \lambda)$ 是 $p$ 作为 $\mathcal{P}(\mathbf{C})$ 中元素的因式分解中的一项。4.14保证 $(x - \bar{\lambda})$ 也作为一项出现在因式分解中，但4.14没有说明这两项出现次数相同，这正是使上述想法成立所需要的。然而，证明绕过了这一点。

在下一个结果中，$m$ 或 $\mathcal{M}$都可能等于 $0$。数 $\lambda_1, \ldots, \lambda_m$ 正是 $p$ 的实零点，因为这些是使下述结果中等式右边等于 $0$ 的仅有的实数值 $x$。

**4.16 实数域上多项式的因式分解**

设 $p \in \mathcal{P}(\mathbf{R})$ 是一个非常数多项式。则 $p$ 有唯一的（因式顺序除外）如下形式的因式分解

$$p(x) = c(x - \lambda_1) \cdots (x - \lambda_m)(x^2 + b_1 x + c_1) \cdots (x^2 + b_{\mathcal{M}} x + c_{\mathcal{M}}),$$

其中 $c, \lambda_1, \ldots, \lambda_m, b_1, \ldots, b_{\mathcal{M}}, c_1, \ldots, c_{\mathcal{M}} \in \mathbf{R}$，且对每个 $k$ 有 $b_k^2 < 4c_k$。

**证明** 首先我们将证明所需因式分解的存在性，之后我们证明唯一性。

将 $p$ 看作 $\mathcal{P}(\mathbf{C})$ 中的元素。如果 $p$ 的所有（复）零点都是实数，则由4.13我们有所需的因式分解。因此假设 $p$ 有一个零点 $\lambda \in \mathbf{C}$ 且 $\lambda \notin \mathbf{R}$。由4.14，$\bar{\lambda}$ 也是 $p$ 的零点。因此我们可以写成

$$
\begin{aligned}
p(x) &= (x - \lambda)(x - \bar{\lambda})q(x) \\
&= (x^2 - 2(\operatorname{Re} \lambda)x + |\lambda|^2)q(x)
\end{aligned}
$$

其中 $q \in \mathcal{P}(\mathbf{C})$ 的次数比 $p$ 的次数低两次。如果我们能证明 $q$ 有实系数，则利用对 $p$ 次数的归纳法就完成了这个结果的存在性部分。

为了证明 $q$ 有实系数，我们从上式中解出 $q$，得到

$$q(x) = \frac{p(x)}{x^2 - 2(\operatorname{Re} \lambda)x + |\lambda|^2}$$

对所有 $x \in \mathbf{R}$ 成立。上式意味着 $q(x) \in \mathbf{R}$ 对所有 $x \in \mathbf{R}$ 成立。将 $q(x) = a_0 + a_1 x + \cdots + a_{n-2}x^{n-2}$，其中 $n = \deg p$ 且 $a_0, \ldots, a_{n-2} \in \mathbf{C}$，我们有

$$0 = \operatorname{Im} q(x) = (\operatorname{Im} a_0) + (\operatorname{Im} a_1)x + \cdots + (\operatorname{Im} a_{n-2})x^{n-2}$$

对所有 $x \in \mathbf{R}$ 成立。这意味着一 $\operatorname{Im} a_0, \ldots, \operatorname{Im} a_{n-2}$ 都等于 $0$（由4.8）。因此 $q$ 的所有系数都是实数，正如所需。因此所需的因式分解存在。

现在转向我们因式分解的唯一性问题。形式为 $x^2 + b_k x + c_k$（其中 $b_k^2 < 4c_k$）的 $p$ 的因式可以唯一地写成 $(x - \lambda_k)(x - \bar{\lambda}_k)$ 且 $\lambda_k \in \mathbf{C}$。稍加思考就能看出，$p$ 作为 $\mathcal{P}(\mathbf{R})$ 中元素的两种不同因式分解会导致 $p$ 作为 $\mathcal{P}(\mathbf{C})$ 中元素的两种不同因式分解，这与4.13矛盾。$\blacksquare$
