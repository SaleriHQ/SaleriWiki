## 摘要

循环是编程中最复杂的方面之一。本章讨论循环类型选择、循环控制和循环创建技术。核心观点：保持循环简单，将循环视为黑盒，循环索引应清晰命名。

## 循环类型选择

**循环类型对比**：

| 类型 | 灵活性 | 测试位置 | 适用场景 |
|------|--------|----------|----------|
| for | 固定次数 | 开始 | 已知迭代次数 |
| while | 灵活 | 开始 | 未知次数 |
| do-while | 灵活 | 末尾 | 至少执行一次 |
| foreach | 固定 | 开始 | 遍历容器元素 |

## 进入循环

**原则**：
- 仅从一个位置进入循环
- 初始化代码直接放在循环之前
- 使用 `while(true)` 表示无限循环
- for 循环头只放循环控制语句

**示例**：
```cpp
// 好：循环控制清晰
for (int i = 0; i < MAX; i++) {
    // 只做循环相关的事
}

// 坏：将非控制语句放入循环头
for (inputFile.MoveToStart(), count = 0; !inputFile.EndOfFile();
     recordCount++) {  // recordCount 不是循环控制
    inputFile.GetRecord();
}
```

## 处理循环体

**原则**：
- 使用 { } 括起循环体
- 避免空循环
- 将内部事务工作放在开始或末尾
- 每个循环只执行一个功能

**内部事务工作**：
```cpp
while (!inputFile.EndOfFile()) {
    inputFile >> inputString;
    names[nameCount] = inputString;
    // ...
    nameCount++;        // 内部事务
    totalLength += inputString.length();  // 内部事务
}
```

## 退出循环

**原则**：
- 确保循环终止
- 终止条件明显
- 不要为终止而修改 for 循环索引
- 避免依赖循环索引最终值
- 考虑使用安全计数器

**坏示例**：
```cpp
for (int i = 0; i < 100; i++) {
    if (...) {
        i = 100;  // 坏：玩弄索引
    }
}

// 好
bool found = false;
for (int i = 0; i < MAX && !found; i++) {
    if (...) {
        found = true;
    }
}
```

## 循环变量

**命名建议**：
- 简单循环用 i, j, k
- 嵌套/复杂循环用有意义名称
- 避免索引串扰

**示例**：
```java
// 坏
for (int i = 0; i < numPayCodes; i++) {
    for (int j = 0; j < 12; j++) {
        for (int k = 0; k < numDivisions; k++) {
            sum += transaction[j][i][k];
        }
    }
}

// 好
for (int payCodeIdx = 0; payCodeIdx < numPayCodes; payCodeIdx++) {
    for (int month = 0; month < 12; month++) {
        for (int divisionIdx = 0; divisionIdx < numDivisions; divisionIdx++) {
            sum += transaction[month][payCodeIdx][divisionIdx];
        }
    }
}
```

## 提前退出

**break**：
- 减少嵌套
- 多 break 是警告信号
- 文档化 break 条件位置

**continue**：
- 在循环顶部测试后使用
- 避免在中间或末尾使用

## 从内到外创建循环

**步骤**：
1. 用字面量写出循环体
2. 替换字面量为变量
3. 用循环包围代码
4. 检查依赖循环索引的变量

**示例**：
```vb
' 步骤1：字面量
rate = table[ ][ ]
totalRate = totalRate + rate

' 步骤2：具体索引
rate = table[census.Age][census.Gender]
totalRate = totalRate + rate

' 步骤3：加循环
For person = firstPerson to lastPerson
   rate = table[census[person].Age][census[person].Gender]
   totalRate = totalRate + rate
End For

' 步骤4：初始化
totalRate = 0
For person = firstPerson to lastPerson
   rate = table[census[person].Age][census[person].Gender]
   totalRate = totalRate + rate
End For
```
