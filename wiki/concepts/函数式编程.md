## 定义

函数式编程（Functional Programming）是一种编程范式，其中程序由一系列函数组成，函数接收输入并产生输出，不产生副作用。

## 核心特征

| 特征 | 说明 |
|------|------|
| **函数是一等公民** | 函数可以作为参数、返回值、赋值给变量 |
| **表达式求值** | 程序 = 表达式，而非一系列指令 |
| **不可变数据** | 数据创建后不可修改 |
| **无副作用** | 函数调用不改变外部状态 |

## Haskell 的函数式特性

```haskell
-- 函数作为值
applyTwice f x = f (f x)
applyTwice (+3) 5  -- = 11

-- 组合函数
square x = x * x
addOne = (+1)
composed = square . addOne
composed 3  -- = 16
```

## 优势

1. **等式推理** - 可以像代数一样替换证明
2. **易于并行** - 无共享状态，可安全并行执行
3. **易于测试** - 无依赖外部状态
4. **可组合性** - 小函数组合成复杂系统
