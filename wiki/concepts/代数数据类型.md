# 代数数据类型 (Algebraic Data Types, ADT)

## 定义

代数数据类型（ADT）是 Haskell 中定义自定义类型的核心机制。使用 `data` 关键字将已存在的基本类型组合成新的类型。与类型同义词（`type`）不同，ADT 创建的是全新的、与现有类型有本质区别的类型。

## 基本语法

```haskell
-- type 同义词：仅是别名，完全等价
type CustomerID = Int
type ReviewBody = String

-- 代数数据类型：全新类型，与其他类型有本质区别
data CustomerID = CustomerID Int           -- 新类型！
data ReviewBody = ReviewBody String         -- 新类型！
```

**值构造器**（Value Constructor）：等号右侧的 `CustomerID`、`ReviewBody` 是**值构造器**，用于创建该类型的值。

## 结构体风格

```haskell
-- BookInfo: 书的信息
--     Book   = 值构造器（与类型同名）
--     Int    = 书号
--     String = 书名
--     [String] = 作者列表
data BookInfo = Book Int String [String]
              deriving (Show)
```

## 记录语法（Record Syntax）

使用大括号指定字段名，Haskell 自动生成**访问函数**：

```haskell
data BookInfo = Book
  { bookId      :: Int
  , bookTitle   :: String
  , bookAuthors :: [String]
  } deriving (Show)

myBook = Book
  { bookId      = 12345
  , bookTitle   = "Real World Haskell"
  , bookAuthors = ["Bryan", "John", "Don"] }

bookId myBook          -- = 12345
bookTitle myBook       -- = "Real World Haskell"
```

## 枚举类型

枚举是 ADT 的特例——只有零参数构造器：

```haskell
-- 枚举：每种花色是一个构造器
data Suit = Hearts | Diamonds | Clubs | Spades
          deriving (Show)

-- 结合参数
data Rank = Rank Int          -- 普通等级
          | Jack | Queen | King | Ace  -- 特殊等级
          deriving (Show)
```

## 参数化类型（多态 ADT）

ADT 可以接受类型参数：

```haskell
-- Maybe a：要么是 Nothing，要么是 Just a
data Maybe a = Nothing | Just a
             deriving (Show)

safeHead :: [a] -> Maybe a
safeHead []    = Nothing
safeHead (x:_) = Just x

safeHead [1,2,3]  -- = Just 1
safeHead []       -- = Nothing

-- Either a b：要么是 Left a，要么是 Right b
data Either a b = Left a | Right b
                deriving (Show)
```

### MessageTree：二叉搜索树

来自 [[wiki/sources/cis194-hw2]] 的实战例子，展示了 ADT 在实际项目中的应用：

```haskell
-- 二叉搜索树：按时间戳排序
data MessageTree = Leaf                          -- 空树
    | Node MessageTree LogMessage MessageTree     -- 递归构造

-- 按时间戳插入：递归遍历左/右子树
insert :: LogMessage -> MessageTree -> MessageTree
insert msg Leaf = Node Leaf msg Leaf
insert msg@(LogMessage _ ts _) (Node left nMsg@(LogMessage _ nTs _) right)
    | ts < nTs   = Node (insert msg left) nMsg right
    | otherwise  = Node left nMsg (insert msg right)
insert _ tree = tree  -- Unknown 消息直接丢弃

-- 中序遍历：左 → 根 → 右
inOrder :: MessageTree -> [LogMessage]
inOrder Leaf = []
inOrder (Node left msg right) = inOrder left ++ [msg] ++ inOrder right
```

## 递归类型

ADT 可以递归定义——列表本身就是递归 ADT 的经典例子：

```haskell
-- 这是标准库中 [a] 的简化定义
data List a = Empty | Cons a (List a)
            deriving (Show)

-- 自定义的自然数
data Natural = Zero | Succ Natural
             deriving (Show)

toNatural :: Int -> Natural
toNatural 0 = Zero
toNatural n = Succ (toNatural (n - 1))

fromNatural :: Natural -> Int
fromNatural Zero     = 0
fromNatural (Succ n) = 1 + fromNatural n
```

## 与模式匹配的结合

ADT 与[[wiki/concepts/模式匹配]]是天作之合：

```haskell
data Shape = Circle Double        -- 圆（半径）
           | Rectangle Double Double  -- 矩形（宽，高）
           deriving (Show)

area :: Shape -> Double
area (Circle r)        = pi * r * r
area (Rectangle w h)   = w * h

isRound :: Shape -> Bool
isRound (Circle _)     = True
isRound (Rectangle _ _) = False
```

## derive 自动派生

`deriving` 让 Haskell 自动为某些类型类生成实例：

| 派生类型类 | 用途 |
|-----------|------|
| `Show` | 转换为字符串（`show` 函数） |
| `Eq` | 相等性判断（`==` `/=`） |
| `Ord` | 排序比较（`<` `>` 等） |
| `Read` | 从字符串解析（`read` 函数） |
| `Enum` | 枚举类型，可用于列表范围 |
| `Bounded` | 有界类型（有最小/最大值） |

```haskell
data Color = Red | Green | Blue
           deriving (Show, Eq, Ord)
-- 自动获得: Red == Red      = True
--          Red < Blue       = True
--          show Red         = "Red"
```
