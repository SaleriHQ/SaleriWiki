## 定义

类型系统是一套规则，用于将值分类为不同的"类型"，并规定类型之间的运算规则。Haskell 拥有静态、强类型系统，配合强大的类型推断。

## Haskell 基础类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `Int` | 机器整数 (~±10^9) | `42` |
| `Integer` | 任意精度整数 | `42^42` |
| `Double` | 双精度浮点 | `3.14` |
| `Bool` | 布尔值 | `True / False` |
| `Char` | 单个字符 | `'a'` |
| `String` | 字符串 | `"hello"` |

## 类型变量 (Polymorphism)

使用类型变量实现泛型：

```haskell
-- 'a' 是类型变量，可以是任意类型
fst :: (a, b) -> a
fst (x, y) = x

length :: [a] -> Int
length [] = 0
length (_:xs) = 1 + length xs
```

## 类型类 (Type Classes)

类型类是 Haskell 的"接口"机制：

```haskell
-- Eq: 可比较相等
(==) :: Eq a => a -> a -> Bool

-- Ord: 可排序
(<) :: Ord a => a -> a -> Bool

-- Show: 可转换为字符串
show :: Show a => a -> String

-- Num: 数值运算
(+) :: Num a => a -> a -> a
```

## 类型推断

Haskell 自动推断类型：

```haskell
-- Haskell 自动推断: f :: Int -> Int
f x = x + 1
```
