# 多态 (Polymorphism)

## 定义

"多态"来自希腊语 πολύμορφος，意为"具有多种形态"。在 Haskell 中，多态意味着代码可以适用于多种类型，而无需为每种类型重复编写。

Haskell 支持两类多态：

| 类型 | 说明 | 示例 |
|------|------|------|
| **参数化多态 (Parametric)** | 类型变量 `a`、`b` 表示任意类型 | `id :: a -> a` |
| **约束多态 (Constrained)** | 类型类约束，限制类型变量范围 | `(+) :: Num a => a -> a -> a` |

## 参数化多态

通过**类型变量**（必须以小写字母开头）实现：

```haskell
-- identity 函数：对任何类型都成立
id :: a -> a
id x = x

-- 长度函数：适用于任意元素类型的列表
length :: [a] -> Int
length []     = 0
length (_:xs) = 1 + length xs

-- 多态列表
data List t = E | C t (List t)
```

**关键原则**：调用者选择类型。写多态函数时，不能根据类型做条件判断——函数必须对所有类型都成立。

## Haskell 的多态 vs 面向对象的多态

| Haskell（参数化多态） | 面向对象（子类型多态） |
|---------------------|---------------------|
| 编译器选择具体类型 | 子类重写父类方法 |
| 无运行时成本 | 通常有虚表查找 |
| 类型变量代表"任意类型" | 继承层次结构 |

## 与 Haskell 类型类的区别

类型类（`Eq`、`Ord`、`Show` 等）提供了**约束多态**，是对参数化多态的补充：

```haskell
-- 参数化多态：类型完全抽象
length :: [a] -> Int

-- 约束多态：类型受类型类约束
sort :: Ord a => [a] -> [a]  -- a 必须是可排序的类型
```
