# 类型推断 (Type Inference)

## 定义

Haskell 能在**不显式标注类型**的情况下自动推断出表达式的类型。

## 示例

```haskell
-- Haskell 自动推断: add :: Num a => a -> a -> a
add x y = x + y

-- GHCi 查看实际类型
ghci> :t add
add :: Num a => a -> a -> a
```

## 工作原理

Haskell 通过以下规则推断类型：

1. **字面量推断**: `3` → `Num a => a`
2. **函数参数推断**: `f x` 中，如果 `f :: a -> b`，则 `x :: a`
3. **约束传播**: `x + y` 要求 `x, y :: Num a`

## 为什么重要

| 优势 | 说明 |
|------|------|
| **减少样板代码** | 不必为每个表达式写类型签名 |
| **类型安全** | 推断仍保证类型正确 |
| **灵活重构** | 改变实现时无需更新类型声明 |

## 来自 [[wiki/sources/real-world-haskell]] 的描述

> "Haskell's type system is unusual in that it is **stateless**: we do not need to declare the type of a variable before we use it."

## 类型推断的边界

```haskell
-- ❌ 错误：类型不匹配
-- "hello" + 5

-- ✅ 正确：显式类型标注
add :: Int -> Int -> Int
add x y = x + y
```

## 多态函数推断

```haskell
-- identity 函数的推断过程
id x = x
-- 推断: id :: a -> a

-- 分析:
-- - x 的类型是 a
-- - 返回值是 x，所以类型也是 a
-- 结果: a -> a
```
