# 单子 (Monad)

## 定义

单子（Monad）是函数式编程中的一种抽象类型，用于**组合计算**，特别是包含**副作用**的计算。它是 Applicative 的超集，提供了一种顺序组合计算的方式。

## 类型签名

```haskell
class Applicative m => Monad m where
    (>>=)  :: m a -> (a -> m b) -> m b     -- bind 操作
    (>>)   :: m a -> m b -> m b            -- 丢弃左值
    return :: a -> m a                      -- 包装值
```

## 核心操作

| 操作 | 符号 | 作用 |
|------|------|------|
| `return` | - | 将纯值包装进 Monad |
| `>>=` | bind | 将 Monad 值传入函数 |
| `>>` | then | 忽略左值，保留右值 |

## Monad 定律

1. **左单位元**: `return a >>= f ≡ f a`
2. **右单位元**: `m >>= return ≡ m`
3. **结合律**: `(m >>= f) >>= g ≡ m >>= (\x -> f x >>= g)`

## Monad 与Applicative

```
Functor     →  (<$>)     映射
Applicative →  (<*>)     包装函数应用
Monad       →  (>>=)     顺序组合（可依赖前一步结果）
```

Monad 不是 Functor 的超集（数学上），但在 Haskell 中 `return` 隐式提供了 Functor。

## do 语法

`>>=` 的语法糖，使 Monad 组合更易读：

```haskell
-- 使用 >>=
result = getLine >>= \name ->
         putStrLn ("Hello, " ++ name)

-- 使用 do 语法
result = do
    name <- getLine
    putStrLn ("Hello, " ++ name)
```

`<-` 绑定从 Monad 中提取值，`return` 退出 do 块。
