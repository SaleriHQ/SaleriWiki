# 模式匹配 (Pattern Matching)

## 定义

模式匹配允许你根据数据的**结构/形状**来定义函数的不同分支。

## 基本语法

```haskell
-- 根据值匹配
describe :: Int -> String
describe 0 = "零"
describe 1 = "一"
describe 2 = "二"
describe n = "其他数字: " ++ show n
```

## 列表模式

```haskell
-- 空列表
sum' :: [Int] -> Int
sum' [] = 0

-- 头部 + 尾部
sum' (x:xs) = x + sum' xs

-- 第一个和第二个元素
firstTwo :: [a] -> Maybe (a, a)
firstTwo (x:y:_) = Just (x, y)
firstTwo _        = Nothing
```

## 元组模式

```haskell
-- 解构元组
addPairs :: (Int, Int) -> Int
addPairs (x, y) = x + y

-- 多参数
swap :: (a, b) -> (b, a)
swap (x, y) = (y, x)
```

## 通配符 `_`

```haskell
-- _ 匹配任何值，但不绑定变量名
head' :: [a] -> a
head' (x:_) = x

-- 不关心第二个参数
f :: a -> b -> a
f x _ = x
```

## @-patterns（as-模式）

匹配模式的同时保留整体变量：

```haskell
baz :: Person -> String
baz p@(Person n _ _) = "The name field of (" ++ show p ++ ") is " ++ n
--  p 绑定了整个 Person，n 只绑定了第一个字段

-- 等价于：
baz (Person n _ _) = "The name field of (" ++ show (Person n undefined undefined) ++ ") is " ++ n
--  ↑ 用 @ 就不必重建整个值
```

### 嵌套模式

可以在一个模式中深入多层结构：

```haskell
checkFav :: Person -> String
checkFav (Person n _ SealingWax) = n ++ ", you're my kind of person!"
checkFav (Person n _ _)          = n ++ ", your favorite thing is lame."
```

## case 表达式

`case` 是模式匹配的表达式形式，可在任意位置使用：

```haskell
describe :: Int -> String
describe n = case n of
    0 -> "零"
    1 -> "一"
    n -> "其他: " ++ show n
```

### 配合 words 解析字符串

```haskell
parseMessage :: String -> LogMessage
parseMessage s = case words s of
    ('E':sev:ts:rest) -> LogMessage (Error (read sev)) (read ts) (unwords rest)
    ('I':ts:rest)     -> LogMessage Info (read ts) (unwords rest)
    ('W':ts:rest)     -> LogMessage Warning (read ts) (unwords rest)
    _                  -> Unknown s
--              ↑ 注意是 ('E':...)，列表用 () 而非 []
```

## 守卫 (Guards)

与模式匹配配合的守卫：

```haskell
describeSign :: Int -> String
describeSign n
  | n < 0     = "负数"
  | n == 0    = "零"
  | otherwise = "正数"
```

## 来自 [[wiki/sources/cis194-hw1]] 的示例

```haskell
-- 递归提取数字
toDigitsRev :: Integer -> [Integer]
toDigitsRev n
  | n <= 0    = []           -- 基准情况
  | otherwise = n `mod` 10 : toDigitsRev (n `div` 10)
                                    -- 递归情况
```

## 最佳实践

| 实践 | 说明 |
|------|------|
| **穷尽检查** | 确保所有情况都被覆盖 |
| **特定到一般** | 匹配顺序从具体到抽象 |
| **使用守卫** | 当需要布尔条件时用守卫 |

## 常见错误

```haskell
-- ❌ 非穷尽匹配，运行时可能出错
firstItem :: [a] -> a
firstItem (x:_) = x
-- 没有处理空列表!

-- ✅ 安全的版本
safeHead :: [a] -> Maybe a
safeHead []    = Nothing
safeHead (x:_) = Just x
```
