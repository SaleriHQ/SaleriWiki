# 递归 (Recursion)

## 定义

递归是函数式编程的核心支柱。在 Haskell 中，递归意味着函数调用自身来解决问题。

## 基本结构

每个递归函数都需要两个部分：

1. **基准情况 (Base Case)**: 最简单的情况，直接返回结果
2. **递归情况 (Recursive Case)**: 将问题分解为更小的子问题

## Haskell 示例

### 阶乘

```haskell
factorial :: Integer -> Integer
factorial 0 = 1                    -- 基准情况
factorial n = n * factorial (n-1)  -- 递归情况
```

### 列表长度

```haskell
length' :: [a] -> Int
length' [] = 0          -- 空列表：基准情况
length' (x:xs) = 1 + length' xs  -- 递归情况
```

### 汉诺塔

```haskell
hanoi :: Integer -> Peg -> Peg -> Peg -> [Move]
hanoi 0 _ _ _ = []                                    -- 0个盘子：无需移动
hanoi n a b c = 
    hanoi (n-1) a c b ++ [(a,b)] ++ hanoi (n-1) c b a
```

## 递归与循环的对比

| 递归 | 循环 |
|------|------|
| 声明式 | 命令式 |
| 自然处理树结构 | 自然处理线性结构 |
| 无需可变状态 | 需要计数器 |

## 递归数据结构的三大操作

来自 [[wiki/sources/cis194-hw2]] 的练习展示了处理递归数据结构的通用模式：

```haskell
data MessageTree = Leaf | Node MessageTree LogMessage MessageTree

-- 1. 插入（insert）：递归找到正确位置
insert :: LogMessage -> MessageTree -> MessageTree

-- 2. 构建（build）：从列表依次插入
build :: [LogMessage] -> MessageTree
build = foldr insert Leaf

-- 3. 遍历（inOrder）：中序遍历得到有序列表
inOrder :: MessageTree -> [LogMessage]
inOrder Leaf = []
inOrder (Node left msg right) = inOrder left ++ [msg] ++ inOrder right
```

> **核心思想**：这三种操作（插入、构建、遍历）构成了处理所有递归数据结构的万能范式。
