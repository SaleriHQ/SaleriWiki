## 目录结构

```
/opt/langlearn/
├── docker-compose.yml      # Docker 编排
├── .env.production          # 生产环境变量
├── nginx/
│   └── nginx.conf          # Nginx 配置
├── backups/                # 数据库备份
│   └── README.md
├── data/
│   ├── books/              # 书籍文件
│   ├── images/            # 用户图片
│   └── exports/           # 导出文件
└── dicts/                  # 词典文件
    └── ecdict.sqlite
```

## 环境变量清单

### 后端环境变量

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `DATABASE_URL` | PostgreSQL 连接 | `postgresql://user:pass@host:5432/db` |
| `REDIS_URL` | Redis 连接 | `redis://host:6379` |
| `JWT_SECRET` | JWT 密钥 | `your-secret-key` |
| `JWT_EXPIRES` | Token 过期时间 | `7d` |
| `STRIPE_SECRET_KEY` | Stripe 密钥 | `sk_live_xxx` |
| `STRIPE_WEBHOOK_SECRET` | Webhook 密钥 | `whsec_xxx` |
| `DICT_PATH` | 词典文件路径 | `/root/dicts/ecdict.sqlite` |
| `UPLOAD_DIR` | 上传目录 | `/opt/langlearn/data` |
| `MAX_FILE_SIZE` | 最大文件大小 | `104857600` (100MB) |

### 前端环境变量

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `NUXT_PUBLIC_API_URL` | API 地址 | `https://api.example.com` |
| `NUXT_PUBLIC_STRIPE_KEY` | Stripe 公钥 | `pk_live_xxx` |
